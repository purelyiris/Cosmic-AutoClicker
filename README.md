**Cosmic AutoClicker**

A lightweight, simple auto clicker that runs directly from a .exe file and is fully open sourced!

To install it, follow these simple steps:
1. Download the AutoClicker.exe file.
2. 2 Run the .exe file, and you're done!
  Configure it to your needs and then hit F6 (if you have not changed the start bind.)

*Note: This has not been tested on any OS aside from Windows 11*

Version: 1.0.0
